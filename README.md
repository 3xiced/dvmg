# DVMG

# Install

## Windows

 - Create virtual environment
```
python -m venv env
```
 - Activate virtual environment
```
.\env\Scripts\Activate.ps1
```
 - Install requirements
```
pip install -r requirements.txt
```
 - Run
```
python app
```

## Linux

 - Create virtual environment
```
python3 -m venv env
```
 - Activate virtual environment
```
source env/bin/activate
```
 - Install requirements
```
pip install -r requirements.txt
```
 - Run
```
python app
```
